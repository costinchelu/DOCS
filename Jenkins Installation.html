Installing Jenkins

The simplest way to install jenkins is by downloading and installing jenkins as a war file.

Go to https://jenkins.io/download/ and choose Generic Java package.

This will download file called jenkins.war. This is a Java web application that runs jenkins within an embedded web server.

To run jenkins, got to the command line and type:

# java -jar jenkins.war

This will run jenkins on port 8080.

You can then access the jenkins console on http://localhost:8080. If you want to run jenkins on a different port, you can specify the port on the command line :

#java -jar jenkins.war --httpPort=<port>

To work with jenkins for this course, we need to have Java, maven and git client installed.

Jenkins Configuration

When you start jenkins a generate admin password is displayed on the console

Start the Jenkins console at http://localhost:<port>

Enter the password generate on the console.

Next Jenkins displays the following screen:

Choose install suggested plugins

Next choose the admin user and password.

Next we need to install the maven plugin

Choose ‘manage Jenkins’ ‘manage plugins’

Click the available plugins tab and search for maven:

Choose the maven integration plugin and choose install without restart

Choose manage Jenkins -> Global tool configuration

In the JDK section add your JDK home and JDK name

Make sure your git section points to the correct location of your git executable:

By default, the git executable should be in the PATH, so that should be OK.

On the Maven section choose add Maven:

Enter your maven home location and give your maven installation a name.

This should complete your Jenkins setup.

We can now work with git and maven in our Jenkins build server.
